Model checkpoint files are not included in this repository.

Place checkpoints directly in this directory:

```text
Model_Checkpoints/
  CelebA.pt
  AFHQ-Cat.pt
```

For PnP-GS, optional gradient-step checkpoints use:

```text
Model_Checkpoints/
  CelebA-GradientStep.pt
  AFHQ-Cat-GradientStep.pt
```
